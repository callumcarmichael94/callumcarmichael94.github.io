Project Details

Holton Lee, a wellbeing discovery centre set in 350 acres of beautiful Dorset countryside, asked for an application that would help visitors understand how to navigate and learn about the centre. This was accomplished by creating a visually friendly geo-location app that guided the user around the centre. When the user arrived at an important location they were given access to various features such as basic and advanced information, photos/videos and social posts from previous users. Holton Lee was extremely impressed with how this application would improve engagement and understanding within the centre.
